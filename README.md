# Chhikara Industries

### Software built for everyone.

Chhikara Industries is a software company focused on building **affordable, accessible, and useful technology for everyone**.

We believe great software shouldn't be limited by expensive subscriptions, complicated systems, or unnecessary barriers. Our goal is to create products that are powerful enough to make a difference while remaining accessible to the people who need them.

---

## What We Build

We create software designed to solve real-world problems and make technology easier to access.

Our work includes:

* 🤖 Artificial Intelligence & Machine Learning
* 💻 Web Applications
* 🛠️ Developer Tools
* 🧩 Browser Extensions
* 📱 Applications & Utilities
* ☁️ Cloud-Based Software
* 🔧 Open-Source Projects

From small tools that make everyday tasks easier to large platforms designed to help people work and learn, we build software with **people at the center**.

---

## Our Philosophy

> **Technology should be accessible to everyone.**

We believe affordability and quality shouldn't have to be opposites.

Our goal is to build software that is:

**Affordable** — Powerful tools shouldn't require an enormous budget.

**Accessible** — Technology should be easy to understand and use.

**Practical** — We build solutions for real problems, not problems that exist only to sell a product.

**Innovative** — We continuously explore better ways to solve problems.

**User-focused** — The people using our software come first.

---

## Our Mission

Our mission is simple:

**Build useful technology. Keep it affordable. Make it accessible to everyone.**

We want to create software that helps students, developers, professionals, businesses, and everyday users get more done without unnecessary costs or complexity.

---

## Open Source

We believe in the power of open-source software and the communities that build it.

Some of our projects are open to the public, allowing developers and creators to explore, contribute, learn, and build upon our work.

## Projects

Our GitHub organization is home to projects across different areas of technology.

We envision a world where powerful software isn't something only a few people can afford.

A world where students can access the tools they need to learn.

Where developers can build without unnecessary barriers.
